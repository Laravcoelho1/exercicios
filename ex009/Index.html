Trabalhando com listas

OL - Ordered lists (Listas Ordenadas)

Listas ordenadas são listas em que a ordem dos itens importa, então mesmo que você coloque um intem depois na lista,
em cima de outros, a numeração vai continuar na ordem certa

EXEMPLO

  1. Acordar
  2. pintar as unhas
  3. Escovar os dentes
  4. Arrumar a casa
  5. Lavar louça
  6. Tomar café da manhã
  7. Assistir a aula
  8. Fazer lições da faculdade

O item "pintar as unhas" Foi adicionado depois, mas por se tratar de uma ordered list, os números em ordem foram corrigidos automaticamente.
O parâmetro "type" em frente a tag ol, permite escolher de que forma essa lista será ordenada

  • 1 - Ordem numérica
  • a - Ordem alfabética minúsculo
  • A - ordem alfabética maiusculos
  • i - Números romanos minúsculo
  • I - Números romanos maiusculos

UL - Unordered Lists (listas não ordenadas)

Agora vamos aprender a fazer uma lista não ordenada, igual a última lista acima.
A unordered list, é formada pela tag "ul" e assim como a ol, usamos a tag "li" para inserir os componentes desta lista.
Observação: A tag "li" não é obrigatório seu fechamento na HTML5.

  • Manteiga
  • Abacaxi
  • Cenoura
  • Maçã
  • Cenoura
  • beterraba

Também é possível personalizar o estilo do marcador com o parâmetro "type", sendo possível utilizar: Square, circle e disk.

DL- Definition list (listas de definição)

Listas de definição são usadas para definir termos, e tem a seguinte hierarquia de tags, sendo a principal que inicia a lista "dl"

Definition term "dt"
Description Definition "dd"

Exemplo:

HTML
Linguagem de marcação para a criação do contéudo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
Java script
Linguagem de programação para criar a interatividade de um site.
