Seletores personalizados (principal)

ID

Função

Caso você determine um estilo para os h1, mas quer que um ou mais h1 fiquem formatados diferentes, além de aplicar um CSS inline, você pode usar seletores personalizados.

lembre-se que você só pode utilizar 1id personalizado por documento html

Como usar?

Basta utilizar a tag id= "nome escolhido por você" em frente ao conteúdo que deseja diferenciar

Exemplo: "h1 id= "principal" >Criado Sites com HTML e CSS (finge que o código está com as chaves)
